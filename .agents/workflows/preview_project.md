To run the project effectively, ensure you run these commands in a terminal that meets the Node.js `>=20` environment requirement.

// turbo-all
1. Navigate to the project directory:
```bash
cd c:\Users\ASUS\CrisisGaurd_OpenEnv\CrisisGaurd
```

2. Start the development server for real-time previewing:
```bash
npm run dev
```

3. Open a browser and navigate to the address shown in your terminal (typically `http://localhost:9002`).

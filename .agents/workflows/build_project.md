To build the project effectively, ensure you run these commands in a terminal that meets the Node.js `>=20` environment requirement.

// turbo-all
1. Navigate to the project directory:
```bash
cd c:\Users\ASUS\CrisisGaurd_OpenEnv\CrisisGaurd
```

2. Install the necessary packages and dependencies:
```bash
npm install
```

3. Type-check the project code:
```bash
npx -y tsc --noEmit
```

4. Build the production package:
```bash
npm run build
```

# **App Name**: CrisisGuard: Misinformation Harm Index

## Core Features:

- Message Analysis: Analyze social media messages for potential harm indicators using a weighted formula.
- Emotional Intensity Detection: Identify emotionally charged words within the message using a predefined list.
- Call to Action Detection: Detect phrases encouraging violence or harmful activities based on a set list of keywords.
- Context Mismatch Check: Allow manual flagging of context mismatches, indicating potential disinformation. An image search tool determines whether or not a given image matches the text of the social media post and adds it as tool input.
- Harm Score Calculation & Display: Calculate a risk score based on emotional intensity, call to action, and context mismatch.  Display on a risk meter that changes color to show level of risk.
- Sample Threat Loader: Add a sample threat for ease of demonstration, populating both the suspicious message input box and auto-checking context mismatch.
- Theme Toggle: Enable users to switch between a default dark mode and a light mode.

## Style Guidelines:

- Dark Mode Primary: Deep Indigo (#4F46E5) for a cyber-security feel.
- Dark Mode Background: Dark gray (#111827), slightly desaturated.
- Dark Mode Accent: Violet (#A78BFA) for interactive elements and highlights.
- Light Mode Primary: Navy (#1E3A8A) for a clean government-dashboard feel.
- Light Mode Background: White (#FFFFFF).
- Light Mode Accent: Sky Blue (#7DD3FC) for interactive elements in light mode.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use simple, clean icons from a library like Font Awesome or Material Icons for navigation and status indicators.
- Sidebar navigation on the left, main content area on the right, ensuring a clear visual hierarchy.
- Subtle transitions for theme changes and loading states.
